# VueJS LDAP Manager

## LDAP Manager Checkpoints | ROAD-MAP

### User & Permissions Management
 - Create
 - Modify
 - Password Reset Screen for End-Users
 - Delete

### DNS Management

 - Create DNS Entries
 - Modify DNS Entries
 - Delete DNS Entries
 - Supported Types should at least be:
    * A
    * AAAA
    * CNAME
    * TXT
    * MX
    * DS
    * NS
    * SOA
